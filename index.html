The Symfony2 Comment Bundle bundle allows you to add a form that will allow you to comment on your content and then to retrieve these comments. All this very easily.

Functionality :

  • Not need a lot of configuration

    • This CommentBundle is very is to configure, without lot of code.

  • Comment and response

    • With this bundle, you can add comment, and you can reply to them. The reply system extends on one level deep to keep better visibility.

  • Protect to comment spam

    • The bundle can protect your comments to spam. For that it use Akismet library. all you need is to create an account on Akismet and add your ID in configuration.
